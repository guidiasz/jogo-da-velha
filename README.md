## 🚀 Como instalar

### Clone este repositório

```bash
git clone https://github.com/guidiasz/jogo-da-velha.git
```

### Acesse a pasta do projeto no terminal

```bash
cd jogo-da-velha
```

### Instale as dependências

```bash
npm install
```

### Inicie o servidor

```bash
 npm run dev
```

Agora basta abrir o link `http://localhost:3000` no navegador 😁
